## 组织结构

``` lua
foodie-dev
├── foodie-common -- 工具类及通用代码模块
├── foodie-monitor -- 基于Spring Boot Admin的微服务监控中心
├── foodie-auth -- 基于Spring Security Oauth2的统一的认证中心
├── foodie-user -- 用户中心
├── foodie-pass -- SSO单点登录
├── foodie-gateway -- 网关服务
├── foodie-admin -- 后台管理系统服务
├── foodie-pay -- 支付平台
├── foodie-portal -- 移动端商城系统服务
├── foodie-demo -- 微服务远程调用测试服务
└── config -- 配置中心存储的配置
```
